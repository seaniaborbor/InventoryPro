## 🛠️ Quick Installation

### Requirements
- PHP 8.2+
- MySQL 5.7+
- Composer

### One-Command Setup

```bash
git clone https://github.com/seaniaborbor/InventoryPro.git
cd InventoryPro
composer install
cp .env.example .env
# Edit .env with your database credentials
php spark migrate
php spark db:seed DatabaseSeeder
php spark serve
```

**Access:** `http://localhost:8080`  
**Login:** `admin@innovativegraphics.com` / `Admin@123`
